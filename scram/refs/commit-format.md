# Commit Format

```
<type>(<scope>): <description>

Co-Authored-By: Claude Opus 4.6 (1M context) <noreply@anthropic.com>
```

**Types:** `feat`, `fix`, `chore`, `refactor`, `test`, `docs`, `style`, `perf`, `ci`, `build`, `revert`

**Scope:** the plugin, package, or module affected (e.g., `scram`, `auth`, `api`). Use the narrowest accurate scope.

**Description:** imperative mood, lowercase, no period. Describe *what* the commit does, not *how*.
